⚠️ 偵測到本地伺服器未連線

「網頁網址長截圖」功能需要 Puppeteer 後端支援。請按照以下步驟啟動本地伺服器：

  1. 使用終端機開啟此專案目錄： 
    cd c:\Marco\Developer\localhosttoolbox
  2. 安裝相依性套件 (僅需一次)： 
    npm install
  3. 啟動本地伺服器： 
    npm start
  4. 啟動成功後，本頁面將會自動偵測連線 (🟢 伺服器已連線)。
